## Running Locally

No build step is required. Clone the repository and open `index.html` directly in a browser:

```bash
git clone https://github.com/mridul2172/pragya-school-website.git
cd pragya-school-website
```

Then open `index.html` in your browser, or serve it with any static file server.

## Deployment

The site is deployed on [Netlify](https://netlify.com), connected directly to this repository. Any change pushed to the `main` branch is automatically built and published to the live site.
