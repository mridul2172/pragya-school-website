# Pragya Convent School — Website

**🔗 Live site: [pragyaconventschoolkatra.netlify.app](https://pragyaconventschoolkatra.netlify.app)**

Official website for **Pragya Convent School**, Katra, Rewa, Madhya Pradesh — an English-medium school offering education from Nursery to Class 8.

## About

This repository contains the source code for the school's public website, built to give parents and the local community easy access to information about admissions, academics, and school life.

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Home page — school overview, facilities, and principal's message |
| `about.html` | School history, mission, and values |
| `academics.html` | Classes offered and teaching approach |
| `admissions.html` | Admission process, age criteria, and required documents |
| `gallery.html` | Photo gallery of school life |
| `contact.html` | Contact details and enquiry form |

## Tech Stack

- **HTML5** and **CSS3** — no frameworks or build tools required
- **Formspree** — handles contact form submissions
- **Netlify** — hosting and continuous deployment from this repository

## Project Structure
## Running Locally

No build step is required. Clone the repository and open `index.html` directly in a browser:

```bash
git clone https://github.com/mridul2172/pragya-school-website.git
cd pragya-school-website
```

Then open `index.html` in your browser, or serve it with any static file server.

## Deployment

The site is deployed on [Netlify](https://netlify.com), connected directly to this repository, and is live at **[pragyaconventschoolkatra.netlify.app](https://pragyaconventschoolkatra.netlify.app)**. Any change pushed to the `main` branch is automatically built and published to the live site.
